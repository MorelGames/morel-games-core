Class: MorelI18n

MorelI18n(locale_loader, available_locales)

Integrates Vue-I18n and Morel games. provides useful methods to load and switch locales.

Constructor

new MorelI18n(locale_loader, available_locales)

Constructs the bridge between the game and Vue-I18n.

For the locale loader, it is recommanded to load the locale using import with a webpack chunk comment, so locales are lazy-loaded instead of bundled in one huge JS file. Example (the end of the webpack comment is altered else the JSDoc comment will break):

locale => import(/* webpackChunkName: "locales-[request]" *\/ "./../locales/" + locale + ".json")
Parameters:
Name Type Description
locale_loader function

A function with the locale as argument, returning a Promise of the catalog for this for this locale.
available_locales object

An object with all available locales for this application. The key is the locale code (e.g. fr) and the value, the locale name in the locale language (e.g. Fran√ßais).

Loads a locale, then switch to this locale.

If a locale with a geographic code is given (i.e. fr_FR) and not found, the base locale (fr) will be tried.

load_locale_from_browser()

Inspects the user browser to determine the best locale.

If there is a locale previously saved in the localStorage, we use that. Else, we look at the browser's exposed language and use the closest locale available, fallbacking to English.
