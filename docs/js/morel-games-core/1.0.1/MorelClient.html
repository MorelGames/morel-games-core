Class: MorelClient

MorelClient(ws_url, protocol)

This class handles communication from/to the game server.

To handle incoming message, create a method named message_in_ then the message's action, replacing dashes by underscores. As example, if a set-uuid message is received, the message_in_set_uuid method will be called. It will get the whole message as argument.

Most of these methods should not be used directly. Instead, use actions defined in the MorelStore module.
